# CEFR – Felles europeisk rammeverk for språk
## Pedagogisk referanse for nivåene A1, A2 og B1
### Kilde: Det felles europeiske rammeverket for språk – utvidet versjon (norsk oversettelse, 2022)

---

## OM RAMMEVERKET

Det felles europeiske rammeverket for språk (CEFR) beskriver språkferdigheter på seks nivåer: A1, A2, B1, B2, C1 og C2. Nivåene grupperes i tre kategorier:

- **Basisbruker:** A1 og A2
- **Selvstendig bruker:** B1 og B2
- **Avansert bruker:** C1 og C2

Deskriptorene er «kan-utsagn» – de beskriver positivt hva en bruker/innlærer **kan gjøre** på hvert nivå, ikke hva de mangler. Hver deskriptor er kort (opptil 25 ord), klart og positivt formulert.

---

## NIVÅ A1 – GJENNOMBRUDD (Basisbruker)

### Overordnet beskrivelse
Nivå A1 regnes for å være det laveste nivået av kreativ språkbruk. Innlæreren har noe mer enn et helt begrenset repertoar av innøvde, situasjonsbetingede fraser som er leksikalsk organisert.

### Hva innlæreren kan gjøre på A1:
- Samhandle på en enkel måte
- Stille og svare på enkle spørsmål om seg selv, om hvor de bor, om folk de kjenner og ting de har
- Komme med og gi respons på enkle utsagn om umiddelbare behov eller svært kjente emner

### Lytte og forstå (A1)
- Kan forstå enkle, korte ytringer når det snakkes langsomt og tydelig
- Kan forstå tall, priser og klokkeslett
- Kan gjenkjenne kjente ord og svært enkle fraser

### Lese (A1)
- Kan forstå korte, enkle tekster med kjente navn, ord og enkle fraser
- Kan forstå enkle skilt og etiketter
- Kan forstå enkle instruksjoner og beskjeder

### Snakke (A1)
- Kan bruke enkle fraser og setninger for å beskrive seg selv og kjente mennesker
- Kan bruke enkle høflighetsfraser: hei, takk, unnskyld
- Kan si ja, nei, unnskyld, takk, beklager
- Kan spørre om og gjøre rede for dag, dato og klokkeslett

### Skrive (A1)
- Kan fylle ut enkle skjemaer med personlige opplysninger: navn, adresse, nasjonalitet, sivilstand
- Kan skrive et enkelt kort/beskjed med lite tekst
- Kan skrive enkle, isolerte ord og fraser

### Ordforråd og grammatikk (A1)
- Begrenset ordforråd bestående av enkeltord og korte fraser
- Svært enkel grammatikk: enkle setningsstrukturer, nåtid og fortid av vanlige verb
- Behov for hyppig repetisjon og bekreftelse

### Oppgavetyper som passer til A1:
- Enkle leseforståelsesoppgaver med korte tekster og bildesstøtte
- Fyll inn blankt med enkle ord (kun ett svaralternativ per blank)
- Match-oppgaver: ord til bilde, ord til definisjon
- Enkle ja/nei-spørsmål
- Ordlister med morsmålsoversettelse
- Sorter ord i kategorier
- Enkle setninger med manglende ord

## NIVÅ A2 – UNDERVEIS (Basisbruker)

### Overordnet beskrivelse
På A2-nivå finner man de fleste deskriptorene som gjelder sosiale funksjoner. Innlæreren kan kommunisere i enkle, rutinepregede situasjoner.

### Hva innlæreren kan gjøre på A2:
- Bruke enkle, hverdagslige høflighetsfraser i hilsener og tiltale
- Hilse på folk, spørre hvordan de har det, og respondere på ny informasjon
- Mestre svært korte sosiale utvekslinger
- Stille og svare på spørsmål om hva man gjør på jobben og i fritiden
- Innhente enkel informasjon om reiser og bruke offentlige transportmidler
- Utføre enkle ærender i butikker, på postkontor og i banker

### Lytte og forstå (A2)
- Kan forstå enkle, hverdagslige beskjeder og kunngjøringer
- Kan forstå enkle instruksjoner om arbeid og daglige gjøremål
- Kan oppfatte innholdet i korte, enkle samtaler om kjente emner

### Lese (A2)
- Kan forstå korte, enkle tekster om kjente emner
- Kan forstå enkle arbeidsinstrukser og beskjeder
- Kan forstå enkle brev og e-poster om kjente temaer
- Kan finne spesifikk, forutsigbar informasjon i enkle hverdagstekster

### Snakke (A2)
- Kan delta i enkle samtaler om kjente emner
- Kan beskrive dagligdagse aktiviteter, rutiner og arbeidsoppgaver
- Kan bruke enkle fraser til å gi informasjon om seg selv og arbeidsplassen
- Kan kommunisere i situasjoner med forutsigbart innhold

### Skrive (A2)
- Kan skrive korte, enkle beskjeder og notater
- Kan fylle ut enkle skjemaer med relevant informasjon
- Kan skrive korte setninger om kjente hverdagsemner
- Kan skrive enkle arbeidsbeskjeder

### Ordforråd og grammatikk (A2)
- Enkelt ordforråd knyttet til hverdags- og arbeidssituasjoner
- Enkel setningsstruktur med koordinering (og, men, fordi)
- Nåtid, fortid og enkel fremtid
- Fremdeles begrenset, men mer fleksibelt enn A1

### A2+ (høyt utviklet A2)
Innlærerne kan delta mer aktivt i samtalen med litt assistanse:
- Innlede, opprettholde og avslutte en enkel, begrenset samtale ansikt til ansikt
- Gjøre seg forstått og utveksle opplysninger om kjente emner i hverdagssituasjoner
- Samhandle forholdsvis utvungent i vante situasjoner (med litt hjelp)
- Fortelle med enkle ord om dagligdagse sider ved sitt eget miljø (jobb, fritid, rutiner)

### Oppgavetyper som passer til A2:
- Leseforståelse med korte fagtekster og multiple choice
- Fyll inn blankt med ordbank
- Sorter setningsdeler i riktig rekkefølge
- Sant/usant-påstander om en tekst
- Enkle dialogoppgaver
- Skriv om egne arbeidsoppgaver med hjelpestrukturer
- Koble ord/begreper til definisjoner eller bilder
- Ordlister med forklaring og morsmålsoversettelse

## NIVÅ B1 – TERSKEL (Selvstendig bruker)

### Overordnet beskrivelse
Nivå B1 kjennetegnes av to hovedelementer:
1. **Evne til å opprettholde samhandling og få fram det man vil** i en rekke situasjoner
2. **Evne til å mestre problemer i hverdagen på en fleksibel måte**

### Hva innlæreren kan gjøre på B1:
- Forstå hovedpunktene i tydelig, standard tale om kjente emner på jobb, skole og fritid
- Klare seg i de fleste situasjoner som kan oppstå under reiser i et område der språket brukes
- Produsere enkel, sammenhengende tekst om kjente emner
- Beskrive erfaringer, hendelser, drømmer og ambisjoner
- Gi korte begrunnelser for meninger og planer

### Lytte og forstå (B1)
- Kan forstå hovedpunktene i tydelig, standard tale om kjente emner
- Kan forstå enkle tekniske opplysninger (instruksjoner, prosedyrer)
- Kan forstå enkle beskjeder og kunngjøringer på arbeidsplassen
- Kan forstå hoveddelen av korte, tydelige meldinger og kunngjøringer

### Lese (B1)
- Kan forstå tekster skrevet i et enkelt, hverdagslig språk
- Kan forstå beskrivelse av hendelser, følelser og ønsker i personlige brev og e-poster
- Kan forstå rutinepregede brev og faktura
- Kan forstå relevante deler av arbeidsinstrukser og HMS-dokumenter
- Kan velge ut og formidle spesifikk, relevant informasjon fra enkle tekster

### Snakke (B1)
- Kan kommunisere i de fleste kjente situasjoner på arbeidsplassen
- Kan forklare problemstillinger kort, selv om det er vanskelig å uttrykke presist
- Kan gi klare instruksjoner om enkle arbeidsprosedyrer
- Kan klage på noe i enkle situasjoner
- Kan be om forklaring eller gjentakelse når noe er uklart

### Skrive (B1)
- Kan skrive enkle, sammenhengende tekster om kjente emner
- Kan skrive enkle rapporter, beskjeder og e-poster om arbeidsrelaterte temaer
- Kan forklare hvorfor noe er et problem med enkle ord
- Kan ta enkle notater for eget bruk

### Ordforråd og grammatikk (B1)
- Tilstrekkelig ordforråd for å kommunisere om kjente emner med omskrivinger
- Kan bruke et bredt spenn av enkelt språk på fleksibelt vis
- Relativt god kontroll over vanlige grammatiske strukturer
- Feil forekommer, men hindrer sjelden kommunikasjon

### B1+ (høyt utviklet B1)
Innlærere på B1+ kan i tillegg til B1-egenskapene:
- Ta imot og gi beskjeder som formidler forespørsler eller forklarer problemer
- Gi konkret informasjon i et intervju/konsultasjon (for eksempel beskrive symptomer)
- Forklare hvorfor noe er et problem
- Beskrive hvordan man gjør noe med detaljerte anvisninger
- Utveksle faktaopplysninger om kjente emner innenfor eget fagfelt

### Oppgavetyper som passer til B1:
- Leseforståelse med lengre fagtekster og åpne spørsmål
- Fyll inn blankt uten ordbank (kontekstuell forståelse)
- Omformuler setninger med samme betydning
- Sorter avsnitt i riktig rekkefølge
- Finn riktig påstand basert på teksten
- Skriv korte svar på arbeidsrelaterte situasjoner
- Diskusjonsoppgaver: hva ville du gjort om...?
- Identifiser feil i setninger (grammatikk/ordforråd)
- Sammensatte ordlister med kontekstuell bruk

## PEDAGOGISKE PRINSIPPER FOR OPPGAVEDESIGN (A1–B1)

### Tekstlengde og kompleksitet
| Nivå | Tekstlengde | Setningslengde | Ordforråd |
|------|-------------|----------------|-----------|
| A1 | 30–80 ord | 5–8 ord | Svært vanlige, konkrete ord |
| A2 | 80–150 ord | 8–12 ord | Hverdagslig og faglig grunnvokabular |
| B1 | 150–300 ord | 10–20 ord | Utvidet fagvokabular med omskrivinger |

### Oppgavetyper og CEFR-nivå
| Oppgavetype | A1 | A2 | B1 |
|-------------|----|----|-----|
| Multiple choice (enkel) | ✓ | ✓ | ✓ |
| Sant/usant | ✓ | ✓ | ✓ |
| Fyll inn blankt (med ordbank) | ✓ | ✓ | – |
| Fyll inn blankt (uten ordbank) | – | – | ✓ |
| Sorter ord til setning | ✓ | ✓ | ✓ |
| Match ord til definisjon | ✓ | ✓ | ✓ |
| Skriv egne svar (korte) | – | ✓ | ✓ |
| Skriv egne svar (lengre) | – | – | ✓ |
| Finn riktig påstand | – | ✓ | ✓ |
| Grammatikkoppgaver | – | ✓ | ✓ |

### Viktige prinsipper
1. **Autentisk kontekst:** Oppgavene skal knyttes til reelle arbeidssituasjoner
2. **Positiv formulering:** Beskriv hva innlæreren KAN gjøre
3. **Gradvis progresjon:** Start med gjenkjenning → forståelse → produksjon
4. **Ordlister:** Inkluder alltid relevante fagord med morsmålsoversettelse
5. **Visuell støtte:** På A1–A2 er bilder og illustrasjoner viktige hjelpemidler
6. **Korte instruksjoner:** Instruksjoner til oppgaver skal være enkle og tydelige
7. **Repetisjon:** Nøkkelord og strukturer skal gjentas på tvers av oppgaver

## EKSEMPLER PÅ KAN-UTSAGN FOR ARBEIDSLIVET

### A1 – Arbeidslivet
- Kan forstå enkle instrukser om grunnleggende arbeidsoppgaver
- Kan lese og forstå enkle skilt og merker på arbeidsplassen
- Kan si fra om umiddelbare behov på jobben

### A2 – Arbeidslivet
- Kan forstå og følge enkle arbeidsinstrukser
- Kan kommunisere med kolleger om hverdagslige arbeidsoppgaver
- Kan forstå enkle HMS-regler og -instrukser
- Kan fylle ut enkle skjemaer på arbeidsplassen

### B1 – Arbeidslivet
- Kan forstå og følge detaljerte arbeidsprosedyrer
- Kan forklare og gi instruksjoner om arbeidsprosesser
- Kan rapportere om arbeidssituasjoner og problemer
- Kan delta i møter om kjente arbeidsrelaterte temaer
- Kan lese og forstå relevante deler av HMS-håndbøker og fagmanualer
